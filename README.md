# Dental Clinic Telegram Bot

Асинхронний Telegram-бот для стоматологічної клініки на **Python + aiogram + SQLite**.

Бот:
- вітає користувача;
- послідовно збирає інформацію про пацієнта та симптоми через **FSM**;
- шукає найближчий вільний слот у SQLite;
- бронює запис;
- не дає медичних порад, а лише допомагає із записом.

## Tech Stack

- Python 3.10+
- aiogram 2.25.1
- sqlite3

Використана API-лінійка **aiogram v2**, бо саме там присутній підхід `aiogram.dispatcher.filters.state`, який ви попросили використати. Це є в офіційній документації aiogram v2.25.1. citeturn370560search1turn370560search3

## Функціонал

### 1. Опитування пацієнта через FSM
Бот по черзі питає:
- ім'я пацієнта;
- що саме болить;
- які симптоми;
- коли болить;
- як болить.

### 2. SQLite база даних
Під час запуску бот:
- створює файл `clinic.db`, якщо його ще нема;
- створює таблицю `appointments`;
- заповнює найближчі 7 днів слотами:
  - 09:00
  - 10:00
  - 11:00
  - 14:00
  - 15:00

### 3. Автоматичний пошук слота
Після завершення опитування бот:
- шукає найближчий вільний слот (`is_booked = 0`);
- показує його користувачу;
- після підтвердження бронює слот.

### 4. Захист від типових проблем
- `/start` перезапускає сценарій;
- `/cancel` скасовує поточний сценарій;
- некоректні відповіді обробляються;
- якщо слот уже зайняли в момент підтвердження, бот повідомить про це.

## Схема таблиці

```sql
CREATE TABLE IF NOT EXISTS appointments (
    id INTEGER PRIMARY KEY,
    date TEXT NOT NULL,
    time TEXT NOT NULL,
    patient_name TEXT,
    symptoms TEXT,
    is_booked INTEGER NOT NULL DEFAULT 0
);
```

## Як отримати токен бота через BotFather

1. Відкрийте Telegram.
2. Знайдіть **@BotFather**.
3. Надішліть команду `/newbot`.
4. Вкажіть назву бота.
5. Вкажіть username бота, який закінчується на `bot`.
6. BotFather надішле вам **API token**.

## Встановлення

### 1. Клонувати репозиторій

```bash
git clone https://github.com/your-username/dental-clinic-telegram-bot.git
cd dental-clinic-telegram-bot
```

### 2. Створити віртуальне середовище

```bash
python -m venv .venv
source .venv/bin/activate
```


### 3. Встановити залежності

```bash
pip install -r requirements.txt
```

## Налаштування токена

### Варіант 1 — через змінну середовища

Linux/macOS:

```bash
export BOT_TOKEN="YOUR_TELEGRAM_BOT_TOKEN"
```

Windows PowerShell:

```powershell
$env:BOT_TOKEN="YOUR_TELEGRAM_BOT_TOKEN"
```

### Варіант 2 — прямо в `bot.py`

Замініть:

```python
API_TOKEN = os.getenv("BOT_TOKEN", "PASTE_YOUR_BOT_TOKEN_HERE")
```

на ваш токен.

## Важливе обмеження

Цей бот **не є медичним консультантом**. Він лише збирає симптоми у текстовому вигляді та створює запис на прийом. Це відповідає вашій вимозі не давати медичних порад.
